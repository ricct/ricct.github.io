Cloud Database Connectivity (CDBC)

Cdbc (Cloud Database Connectivity) is a .NET library which can help windows native application or web application connect to cloud database.

It consists of Cdbc Client and Cdbc Service. Cdbc Client connects to Cdbc Service, which can provide proxy access to the database. Cdbc Service uses wcf architecture so IIS on Windows instance, or other service that can run wcf on the cloud is needed.

Benefits and features:
  • High security
  • Easy development
  • High performance

Cdbc Client has the same syntax as ODP.net, SqlClient etc. User doesn't do any development on CDBC service.

Simple Data Integrator (SDI)

Simple Data Integrator (SDI) is a data integration tool (ETL) , which extracts data from the source system, converts and processes it according to business requirements, and loads the processed data into a target system.

Main functions
  • Transfer data from the source DB table, view, query to target DB table.
  • Import data from CSV, EXCEL , JSON file into the target database table.
  • Execute SQL and Procedure .
  • Workflow
  • Database availability monitoring
  • Scheduler

MyFX.jp Web Service

MyFX is an all-in-one web service that elevates your FX trading experience to new heights. It offers an intuitive visual display of trading performance data for deeper insights, along with innovative FX simulation and backtesting tools to enhance your trading strategy.

Available only in Japan.
