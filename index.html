Welcome to RiCCT

クラウドテクノロジーラボは、最先端のクラウド技術の研究、開発、応用に焦点を当てます。
クラウドコンピューティング、ビッグデータ処理、人工知能、セキュリティなど、幅広い分野にわたるクラウド関連の技術に深く関与しています.

クラウドコンピューティング

インフラストラクチャの効率的な利用、サービスのスケーラビリティ向上など、クラウドコンピューティングの基盤技術に関する研究

ビッグデータとデータ解析

大規模なデータセットの処理、分析、活用に関する技術的な課題への取り組み。

クラウド上での機械学習モデルの開発、ディープラーニングの進化、AIエコシステムの構築など。

Cloud Database Connectivity (CDBC)

CDBC（Cloud Database Connectivity）は、クラウドデータベースに接続するための.NETライブラリです。 このライブラリを使用することで、Windowsアプリケーションがスムーズにクラウドデータベースに接続できます。

CDBCは、CDBCクライアントとCDBCサービスで構成されています。CDBCクライアントはCDBCサービスに接続し、データベースへのプロキシアクセスを提供します。 CDBCサービスはWCFアーキテクチャを使用するため、Windowsインスタンス上のIISまたはクラウド上でWCFを実行できる他のサービスが必要です。

特徴：
  • 堅固なセキュリティ対策
  • 簡単な開発
  • 高いパフォーマンス

CDBCクライアントはODP.netやSqlClientと同じ構文を持ちます。ユーザーはCDBCサービスの開発を行う必要がありません。

Simple Data Integrator (SDI)

Simple Data Integrator (SDI) は、データベース間でデータを連携し、変換、抽出、ロード（ETL）を実行するためのソフトウェアです。 SDIはデータの移動と変換プロセスを自動化し、異なるデータソース間でデータの一貫性を確保します。

主要機能
  • RDB間データ連携
  • RDBとファイル（CSV,Excel,JSON,XML）間データ連携
  • RDBとWeb Service , Web API 間データ連携
  • SQLクエリやプロシージャの実行
  • ワークフロー機能
  • データベースの可用性監視
  • スケジューラー

MyFX.jp Web Service

MyFXは、あなたのFX取引を新しい次元に引き上げるためのオールインワンWEBサービスです。取引実績データをわかりやすく視覚的に表示し、トレードの洞察を一層深めるとともに、革新的なFXシミュレーションとバックテストツールを提供して、より戦略的なトレードが可能となります。

主要機能
  • ユーザーフレンドリーな多機能ダッシュボード
  • 自動売買証拠金シミュレーション
  • バックテスト機能で戦略を評価

MyFXは、複数の証券会社の取引実績データを一元管理し、分散された情報をシームレスに統合し、ユーザーフレンドリーなダッシュボードに表示します。
未来のFX取引に備え、証拠金シミュレーションとバックテスト機能を提供し、トレーダーに確かな戦略を築く手助けを致します。リスクを最小限に抑えて戦略を洗練しましょう。
